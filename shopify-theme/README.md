# Signs 'N More — Shopify Theme

Bold-industrial Shopify Liquid theme converted from the React/TanStack site.

## Install

1. Zip this folder so `layout/`, `templates/`, `sections/`, etc. sit at the root of the zip.
2. In Shopify admin: **Online Store → Themes → Add theme → Upload zip file**.
3. Click **Customize** to swap images, edit copy, and reorder sections.

## Editable from the Theme Editor

- Header / footer (phone, email, address, Instagram via Theme Settings → Brand)
- Hero (heading, subheading, CTA, image)
- Services, Process, Reviews, Gallery, Stats (blocks)
- Contact form (uses Shopify's built-in contact form — submissions go to your store email)

## Notes

- Add product images via the Theme Editor — no images are bundled.
- `templates/product.liquid`, `collection.liquid`, `cart.liquid` are included so the store works once you add products.
